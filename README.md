# Simple Express API Endpoint with Socket.IO

## Technical Stack

- Express
- Socket.io
- zod for validation

## Features

- TypeScript
- Prettier
- ESLint

## Getting Started

### Install Packages

```bash
$ npm install
```

### Configuration

Create `.env` file and put `PORT` variable. 

If `PORT` is not specified, it will use `3000` by default.

### Run dev server

```bash
$ npm run dev
```
